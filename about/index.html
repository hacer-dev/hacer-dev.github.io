hacer-dev studio

hacer는 스페인어로 "만들다"라는 뜻입니다. 필요한 도구는 직접 만들어서 쓰자는 마음으로 시작했고, 그 결과물이 누군가에게도 도움이 되길 바라며 공개합니다.

🛠 하는 일

  • 데스크탑 유틸리티 개발
  • 기존 앱 기능 수정/개선
  • 새로운 아이디어 프로토타이핑

⚙ 기술 스택

  • 언어: Go, C++, JavaScript
  • GUI: Wails (Go + WebKit), ImGui, HTML/CSS/JS
  • 배포: Linux, Windows
