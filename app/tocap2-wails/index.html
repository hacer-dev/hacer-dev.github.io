toCap²

파일명 대문자 변환기 · Wails 네이티브 앱

⬇ Windows 64bit ⬇ Linux 64bit
버전
1.0.0
최종 업데이트
2026-06-06
기술 스택
Go + Wails + WebKit2GTK

주요 기능

  • 폴더 선택 및 드래그드롭 지원
  • 패턴 필터 (*.txt, *.mp4 등)
  • 하위폴더 포함 전체 탐색
  • 개별 선택 / 전체 선택 / 패턴 일괄 선택
  • 되돌리기 가능 (변환 취소)
  • Wails 네이티브 창 (WebKit 기반 렌더링)

사용 방법

  • 실행하면 현재 폴더의 파일 목록이 표시됩니다
  • 필터에 *.txt 를 입력하고 [선택]을 누르면 .txt 파일만 선택됩니다
  • 변환할 글자 수를 슬라이더로 조절합니다
  • [시작]을 누르면 파일명이 변경됩니다
  • [되돌리기]로 원복하거나 [완료]로 확정합니다
